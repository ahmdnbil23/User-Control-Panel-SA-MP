# SA-MP / Open.mp User Control Panel (UCP)

## 🎯 Purpose / Tujuan
---
**English:**
The purpose of sharing this repository is to contribute to the SA-MP and open.mp community, especially for those who are building servers but facing challenges in developing web-based systems. I hope this UCP project can serve as a reference, a learning resource, or a solid foundation that fellow developers can further enhance and adapt for their own projects.

**Bahasa Indonesia:**
Tujuan saya membagikan repository ini adalah untuk berkontribusi kepada komunitas SA-MP dan open.mp, khususnya bagi para pengembang yang sedang membangun server namun menghadapi kendala dalam pembuatan sistem web. Saya berharap hasil karya UCP ini dapat menjadi referensi, sarana belajar, atau fondasi dasar yang dapat dikembangkan lebih lanjut oleh rekan-rekan developer lainnya demi kemajuan komunitas.

---

## ✨ Features / Fitur Utama
*   **Account Integration:** Terintegrasi langsung dengan database MySQL server.
*   **Character Stats:** Menampilkan informasi detail karakter secara real-time.
*   **Secure System:** Dilengkapi dengan sistem login dan proteksi dasar.
*   **Modern UI:** Tampilan yang bersih dan responsif untuk desktop maupun mobile.

## 🛠️ Tech Stack
*   **Language:** PHP / JavaScript
*   **Database:** MySQL / MariaDB
*   **Frontend:** Bootstrap / CSS Custom

## 🚀 How to Use / Cara Penggunaan
1. Download repository ini ke web server kamu.
2. Download PHPMailer dan taruh di directory web ucp kamu (PHPMailer-6.10.0 Recomendation or new)
3. Konfigurasikan database di setiap php.
4. Import file `.sql` yang disediakan ke database MySQL kamu atau buat sesuai yang di butuhkan di dalam file php.
5. UCP siap digunakan!
